Uso de los comandos basicos de linux


Antes de empezar quiero explicar que los comandos son instrucciones que uno le da al computador por medio de un terminal o consola de comandos, de esta manera se puede administrar de forma mas dinamica los sistemas. Por ejemplo en linux se pueden configurar infinidades de cosas por medio de comandos, como por ejemplo, los servidores ssh, ftp, servidores http, bases de datos, directorios, archivos, entre muchas mas cosas.

Se que muchas personas pueden pensar que el uso de estas terminales es bastante complejo y tienen razon, muchas veces el uso de la terminal se puede volver bastante complejo la verdad, pero siempre se comienza por algo basico no?, hoy veremos como utilizar los comandos basicos de linux.

cd :

Bueno vamos a ver un comando que yo considero escencial en todo sistema operativo
cd
Si este es el comando y sus siglas significan 'change directory' y como su nombre lo dice nos sirve para entrar o cambiar de directorios. Su uso es bastante sencillo :
cd midirectorio
cd : es el comando
midirectorio : podria ser el directorio donde queremos entrar
Aqui te dejo un pequeño ejemplo de como se veria desde un terminal.



Como pudes ver use el comando cd para entrar dentro de la carpeta Documents.

ls :

Su significado es 'list', y funciona para listar archivos, ya sean carpetas, archivos de texto, ejecutables, TODOS. Usarlo tambien es sencillo solamente se escribe el comando asi tal cual ls.



como puedes apreciar al escribir ls y dar enter se listaron todos los archivos dentro de la carpeta Documents.
Puedes listar archivos ocultos con la opcion ls -a , ve y explora ;)

pwd

Su funcion principal es decirte en que directorio te encuentras, yo diria que es de gran ayuda cuando te pierdes entre los directorios, su uso es el siguiente:

pwd

Y su salida es esta:

/home/usuario/Documents

Obviamente la ruta puede variar dependiendo de en que directorio estes ubicado.

cat

Como el nombre del gato, este comando funciona para visualizar archivos de texto, su uso tambien es basico, es el siguiente:

cat miarchivo

cat es el comando y miarchivo es tu archivo de texto, de esta manera tan sencilla podras ver todo tipo de archivo de texto plano, no solamente archivos de texto.

rm

Este comando viene del nombre rmdir y funciona para borrar archivos, su uso es el siguiente

rm -r archivo

rm es el comando y el caracter -r es para especificar que se va a borrar un archivo de forma permanente- Solo te pido que tengas cuidado con este comando es tu directorio raiz, cuidado una trastada.

mv

Su nombre significa move y funciona para mover archivos de un lado a otro, su uso es el siguiente:

mv archivo destino

mv es el comando, donde dice archivo es el archivo que se va a mover, y destino es el lugar donde se movera el archivo. Tambien puedes cambiarle el nombre a un archivo con mv, poniendo el comando, el nombre original, seguido del nombre que le quieres poner.
