Nueva etiqueta html <py-script> </py-script>

21/may/2022

Si en estos dias acaba de estar disponible la nueva etiqueta <py-script> que te permite ejecutar codigo python desde html5.
Todo lo que pongamos entre esa etiqueta se ejecutara, basicamente tiene el mismo funcionamiento que la etiqueta <script> solo que no es para ejecutar codigo de javascript, sino codigo python. Aqui te dejo la pagina oficial de este gran proyecto que porsupuesto beneficiara a miles de desarrolladores.

Anaconda en una publicacion de blog reciente explica. "PyScript tiene como objetivo brindar a los usuarios un lenguaje de programacion de primera clase que tenga reglas de estilo consistentes, sea mas expresivo y mas facil de aprender".

Obviamente esta nueva funcion traera nuevas ideas y proyectos cientificos.
Pero para poder usar py-script debemos poner algo en la etiqueta <head></head> de nuesto codigo html. Si nos vamos a la pagina oficial de py-script veremos una seccion que dice install, si tocamos en ella nos aparecera lo siguiente:

<link rel="stylesheet" href="https://pyscript.net/alpha/pyscript.css"/>
<script defer src="https://pyscript.net/alpha/pyscript.js"</script>



Para probar escribi este codigo simple en html, ahora vamos a probarlo en el navegador...


Y como puedes ver se ejecuta perfectamente. Una desventaja que le veria a esta nueva funcion es que es mucho mas lento que javascript, por ahora javascript llevara la delantera, pero esperemos que pythonscript mejore para bien.
