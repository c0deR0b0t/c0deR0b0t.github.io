Comandos Basicos de linux

Se que a muchos cuando escuchan la palabra comandos o alguna palabra asi por el estilo
se asustan y no quieren seguir aprendiendo ya sea porque lo ven muy dificil o simplemente
pereza de aprender, pero bueno en este tema te mostrare algunos comandos de lo mas basico. Bueno primero lo primero, linux es un sistema operativo creado por Linus Tolvars y es un
sistema operativo OPENSOURCE o como se diria en español codigo libre, pronto profundizare mas
en este tema. Ahora que es un terminal, es una interfaz o una ventana donde le das ordenes a la computadora
a travez de comandos o tambien se le podrian llamar intrucciones, te puede servir para ejecutar
programas, configurar los puertos, los servicios de computador, controlar firewall, controlar
procesos, manejar carpetas y archivos entre muchas otras infinidades de cosas.



cd : Este comando funciona para navegar entra carpetas o directorios, o simplemente se puede usar
para viajar a una ruta especifica, su nombre cd significa ChangeDirectory o algo como cambiar de
directorio en el idioma español, su uso es el siguiente:

cd MiCarpet

Al hacer esto estariamos entrando en la carpeta 'MiCarpet', tambien este comando se usa para retroceder
desde un directorio. Digamos que estamos en /home/MiCarpet y queremos ir al directorio home que esta
una carpeta atras, para ello escribiriamos:

cd ../

Y con esto estariamos en /home/. Facilisimo verdad xd, ok no, sigamos....

El siguiente comando igualmente al anterios es facil de utilizar, este comando es
ls : Funciona para ver los archivos(de cualquier tipo) dentro de un directorio o carpeta
su uso basico seria el siguiente:
ls
Y nos arrojara esta salida:

Descargas Escritorio Musica Publico
Documentos Imagenes Plantillas Videos

Me aparecio esta salida porque ahora mismo estoy en /home/marc/, osea mi directorio home, que pronto un bloc
explicando el uso de las carpetas del directorio raiz. Bueno ls siginifica 'list' y funciona como ya dije
para ver los archivos dentro de una carpeta, pero digamos que hay un archivo oculto en el directorio
donde hicimos el comando 'ls', no nos aparece, les recuerdo que los archivos ocultos siempre llevan un punto
adelante, por ejemplo '.archivo.txt', entonces con simplemente 'ls' no podremos ver el archivo oculto
tendremos que agregarle algo mas, en este caso

ls -a

Y me daria esta salida:

. .config .gtkrc-2.0 .mozilla .Xauthority
.. Descargas .gtkrc-xfce Música .xsession-errors
.bash_history .dmrc .ICEauthority Plantillas .xsession-errors.old
.bash_logout Documentos Imágenes .profile
.bashrc Escritorio .linuxmint Público
.cache .gnupg .local Vídeos

 Como podemos ver habian mas archivos de lo que se pensaba, bueno sigamos con el siguiente comando.



mv: Este comando funciona para mover archivos, su nombre viene de la palabra 'move', su uso es bastante
sencillo y simple, lo unico que hay que hacer es lo siguiente:

mv miarchivo /home/

mv : es el comando :v

miarchivo : es el archivo que movere

/home/ : es al lugar donde se movera el archivo

Bueno el siguiente comando es cp : este funciona para copiar archivos y su nombre proviene de la palabra
'copy', su uso al igual que move es facil:

cp miarchivo /home

cp : es el comando :V

miarchivo : es el archivo que copiare

/home/ : es el lugar donde se copiara el archivo

whoami : Este comando funciona para saber quien eres, de donde vienes, tu existencia, tu parte en el universo
ok... hablando enserio este comando es para ver que usuario eres dentro de la maquina. Su uso es el siguiente:

whoami

Si asi solamente, despues de escribirlo y dar enter en el terminal te aparecera tu nombre de usuario, en mi caso
me aparecio marc, ya que es mi nombre de usuario.
En caso de que quiera saber el nombre de tu maquina debes escribir:

hostname

La salida de este comando te dara el nombre de tu maquina.



Ahora una de las cosas mas importantes, como borrar archivos desde el terminal?
rm : Este comando funciona para borrar archivos, su uso es basico.

rm -r archivo.txt

de esta manera estariamos borrando el archivo, otra forma de borrar archivos de forma mas segura
es con el comando shred, su uso es este:

shred -u archivo.txt

El comando shred a diferencia de rm es que en vez de borrar simplemente los archivos, los destruye y seguido los borra.
Te recomiendo tener cuidado con estos dos comandos ya que si eres usuario root y borras una carpeta importante del sistema
tu computador puede petar y te podria tocar reinstalar el sistema operativo.
